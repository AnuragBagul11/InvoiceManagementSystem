# Invoice Management System

A clean and interview-ready **Invoice Management System** built using **ASP.NET Core MVC, Entity Framework Core, SQL Server, Bootstrap, Serilog, and ClosedXML**.

This project covers complete invoice management with **CRUD operations, authentication, filtering, sorting, soft delete, date range filtering, total summary, and Excel export**.

---

##  Features
- User Login Authentication
- Invoice Create
- Invoice Edit
- Invoice Soft Delete
- Invoice List View
- Status Filter
- Date Range Filter
- Sorting by Date
- Sorting by Amount
- Total Amount Summary
- Total Tax Summary
- Excel Export
- Serilog File Logging
- SQL Server Integration
- Entity Framework Core ORM

---

##  Tech Stack
- ASP.NET Core MVC (.NET 10)
- Entity Framework Core
- SQL Server
- Bootstrap
- Serilog
- ClosedXML
- Razor Views
